# Simple Todo App

A clean, minimalist task management web application designed to help users track their daily objectives and stay organized with clear deadlines.

## Functionality

The Simple ToDo App provides a streamlined user interface focusing on clarity and ease of use. Key features include:

Task Creation: Easily add new tasks to your list using the intuitive + Add Todo modal button.

Progress Tracking: Dynamic status text dynamically shows your completion progress (e.g., "Showing X out of Y completed").

Visual Status Indicators: Toggle tasks between incomplete and completed using stylized checkboxes that update in real-time.

Deadline Management: Every task clearly displays its specific due date to help prioritize urgent responsibilities.

Task Removal: Instantly delete tasks from the dashboard with dedicated Delete actions for decluttering.

## Technology

This project focuses on semantic structure, clean presentation, and interactive state management:

HTML5: Used to build a semantic structure for the application container, input elements, and task lists.

CSS3: Powers the modern, centered card layout, using smooth drop shadows, responsive padding, and clear color coding (blue accents for actions, red for deletions).

JavaScript: Handles DOM manipulation, state changes for checking/unchecking tasks, dynamic progress tracking calculations, and task deletion logic.
